O sistema resolve o problema da fragmentação de ativos digitais, gerenciamento de faixas e a organização de portfólios em álbuns.
=====================================================================================================================================================

🚀 **Funcionalidades Principais**

**Gestão de Faixas**:
Cadastro de músicas com título, artista e link do SoundCloud.
Edição de metadados das faixas através de modais dinâmicos.
Exclusão segura de faixas do catálogo.

**Gestão de Álbuns**:
Criação e edição de álbuns com seleção múltipla de faixas.
Visualização organizada de álbuns, permitindo agrupar músicas para portfólios ou pitchings.

**Player Integrado**:
Embed customizado do SoundCloud, configurado para uma experiência clean, sem anúncios, distrações ou elementos visuais pesados.
Controles de Play/Pause e volume integrados diretamente na interface.

**Busca Rápida**:
Sistema de filtragem em tempo real para localizar faixas no catálogo de forma ágil.

**Autenticação**:
Controle de acesso seguro para que cada músico gerencie apenas o seu próprio catálogo.
